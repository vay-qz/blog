# VuePress搭建（一）

  1. 下载安装node.js

  2. 安装并运行

    #设置淘宝镜像
npm install -g cnpm --registry=https://registry.npm.taobao.org

# 全局安装
npm install -g vuepress

# 创建一个markdown文件
echo '# Hello VuePress' > README.md

# 运行
vuepress dev
